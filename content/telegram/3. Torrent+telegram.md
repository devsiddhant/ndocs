title: Torrent + Telegram
---

Using Telegram for file sharing and leeching.

> [!NOTE]
> Telegram allows file uploads up to 2GB (4GB for Premium). Many users use bots to "leech" torrents to Telegram for direct high-speed downloading.

### Leech Bots
*Note: Public leech bots often go down or become paid. These are volatile.*

*   **Mirror/Leech Groups**: Search for "Leech Group" or "Mirror Group" on Telegram. Many communities allow you to send a magnet link and the bot uploads the file to Telegram.
