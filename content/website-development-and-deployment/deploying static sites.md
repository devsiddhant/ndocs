Where to host your website for free.

## Platforms

### [Vercel](https://vercel.com/)
Creators of Next.js.
*   **Best for**: Dynamic sites, Next.js apps. Excellent free tier and GitHub integration.

### [Netlify](https://www.netlify.com/)
The pioneer of the JAMstack.
*   **Best for**: Static sites, effortless drag-and-drop deployment.

### [Cloudflare Pages](https://pages.cloudflare.com/)
Fast, secure, and free.
*   **Best for**: Performance. Runs on Cloudflare's massive global network. **Highly Recommended**.

### [GitHub Pages](https://pages.github.com/)
Host directly from your GitHub repository.
*   **Best for**: Simple project documentation or personal blogs hosted on GitHub.
